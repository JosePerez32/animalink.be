## ✨ What is Animalink?

Animalink is a character-driven AI universe where every conversation shapes the relationship. Bond with:
- 🦢 **Aureliane** — The Enigmatic Archivist
- 🦆 **Duque** — The Loyal Companion  
- 🐕 **Dobert** — The Watchful Guardian
- 🐰 **Milla** — The Threshold Detective

Each character tracks your bond across five stages: **Stranger → Acquaintance → Trusted → Confidant → Bonded**.

## 📁 Project Structure

```
animalink-web/
├── src/
│   ├── app/                 # Next.js routes
│   │   ├── page.tsx         # Homepage
│   │   ├── characters/      # Character pages
│   │   ├── universe/        # Lore & world
│   │   ├── download/        # App download
│   │   ├── premium/         # Pricing plans
│   │   ├── support/         # FAQ & help
│   │   └── contact/         # Contact form
│   ├── components/
│   │   ├── home/            # Hero, Stats, Features
│   │   ├── characters/      # CharacterCard, BondTracker
│   │   ├── ui/              # Button, Pill, SectionTag
│   │   ├── layout/          # Nav, Footer
│   │   └── effects/         # Starfield, AmbientBlob
│   ├── data/                # Character data
│   └── types/               # TypeScript interfaces
└── public/                  # Static assets
```

## 🛠️ Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Static export
npm run build  # (output: dist/)
```
